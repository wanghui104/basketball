# Halfcourt Sketch

一个网页端半场篮球原型，当前目标是先打磨单人控球、运球、移动、投篮轨迹和篮筐反馈。

## 运行

直接用浏览器打开 `index.html` 即可。

## 操作

- `WASD` / 方向键：移动
- `Shift`：冲刺
- `Space`：投篮
- `R`：重置

## 当前设计重点

- 45 度俯瞰半场视角。
- 人物采用极简线条风格：圆头、躯干、线条四肢，无五官。
- 球的硬碰撞只围绕地面、边界、篮板、篮筐、人物躯干和头部展开。
- 四肢主要作为视觉和动作表达，后续可加入手部软判定窗口。
- 用 Canvas 2D 的世界坐标 `x, y, z` 做伪 3D 表现，方便继续扩展动作和碰撞。

## 下一步建议

- 加入投篮按住蓄力和出手点时机。
- 加入急停、后撤步、交叉步动作状态。
- 把手部交互范围独立出来，用于更细的控球、接球、盖帽和抢断判定。
- 细调篮筐碰撞，让空心、打板和弹筐反馈更有区别。
